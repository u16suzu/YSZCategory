YSZCategory
===========

### Installation

Write to podfile.

You have to specify pod version with tag.

```
pod 'YSZCategory', git: 'https://github.com/u16suzu/YSZCategory.git', tag: '0.0.4'
```

Run the command

```
$ pod install
```

### Usage

Import header file

```
#import <YSZCategory/Category.h>
```
